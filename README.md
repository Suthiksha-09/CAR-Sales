# CAR-Sales Dashboard
Developed a dynamic Car Sales Dashboard in Power BI to track and analyze critical KPIs such as total revenue, units sold, profit margins, and monthly sales trends.Implemented data modeling, DAX measures, and interactive visualizations to provide actionable insights and improve business performance monitoring.
